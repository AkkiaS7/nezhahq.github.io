## 使用说明  
+ 获取快捷指令后，打开并编辑快捷指令  
+ 分别在三个文本框中填入 `面板URL`、`API Token`、`服务器ID`  
+ 保存编辑并测试运行，能获取结果则表示设置正确 
+ 修改快捷指令的名称，如修改为：“一号服务器状态”，保存后便可使用Siri指令：“嘿Siri，一号服务器状态”来获取服务器状态  

::: warning  
每个快捷指令只能监控一台服务器，如需要监控多个服务器，请多次复制此快捷指令并分别配置，然后分别修改快捷指令名称，如；一号服务器状态、二号服务器状态等
:::
